# Diffract website

A zero-dependency static landing page for Diffract, ready for GitHub Pages.

## Preview locally

Open `index.html` directly, or run any static file server in this directory.

## Deploy to GitHub Pages

1. Push these files to a GitHub repository.
2. Open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and `/ (root)`, then save.

The site contains no build step and will be published as-is.
